Privacy Policy for AI Coach Assistant

Last updated: 10 November 2025

1. Overview

AI Coach Assistant is a CustomGPT used internally to generate and manage structured training files. It includes an Action hosted on Google Cloud Run that parses and produces .zwo and .fit workout files.

2. Data Collection and Usage

The system processes user prompts and file data solely to generate or interpret workout files. Inputs are sent to the Cloud Run backend only when necessary to execute these actions. No personal, identifying, or behavioral data is collected for analytics or marketing.

3. Data Storage

  • The Action does not store or retain user data.
  • Inputs/outputs are processed transiently in memory and discarded after the response.
  • Google Cloud may keep minimal, temporary infrastructure logs (e.g., timestamps) for reliability; these logs contain no user content or personal data.

4. Data Sharing

No user data is shared with third parties. Data flows only between OpenAI’s platform and the Cloud Run endpoint to fulfill requests.

5. Intended Use

This GPT is for internal use only, not for public distribution or commercial deployment.

6. Security Measures

All backend communication occurs over HTTPS to authenticated endpoints. No credentials or tokens are exposed in client-side requests.
